# kindle-thumbnail-fix #

 ##_A small software to sideload ebooks to kindle without losing cover art thumbnails_

## Work In Progress: ##
The software is currently under development, it may take some times to finally release a fully working version of it, some of the feature in the current pipeline are

1. Kindle Connection Prompt
2. Fixing Calibre Library for Kindle
3. Sideloading Books
